# Freelancer Dataset Analysis


# Dataset Overview

This project analyzes a comprehensive Freelancer Dataset to understand global freelancer profiles, project performance, skill distribution, hourly rates, and experience levels. The goal is to build an interactive dashboard that provides clear insights into freelancer activity, project progress, and market trends.

# Columns Description

Freelancer ID: Unique identifier for each freelancer

Name: Freelancer’s full name

Age: Freelancer’s age

Country: Location information

Experience Years: Total professional experience in years

Skill Set: List of skills

Project Status: Current status of the project (Completed, In Progress, Pending)

Total Projects: Number of completed projects

Average Rating: Average client rating (out of 5)

Hourly_Rate: Hourly charge in USD

# Key Insights

Most freelancers are concentrated in countries like South Korea and Canada.

Experienced freelancers tend to have higher average ratings.

DevOps, UI/UX Design, and Block chain Development are the most popular project categories.
